# 📊 Customer Analytics Excel Dashboard

## 📌 Project Overview

This project presents a **Customer Data Collection and Analysis Dashboard** developed in **Microsoft Excel** for a growing startup. The solution demonstrates how customer information collected from multiple sources can be consolidated into a single workbook, analysed using Excel formulas, PivotTables, PivotCharts, and slicers, and presented through an interactive dashboard.

The dashboard helps businesses monitor customer behaviour, evaluate sales performance, measure customer satisfaction, and identify opportunities for business growth.

---

## 🎯 Project Objectives

- Design a structured customer data collection template.
- Consolidate customer data from multiple sources into one Excel workbook.
- Analyse customer information using Excel formulas and PivotTables.
- Develop Key Performance Indicators (KPIs) to support business decision-making.
- Create an interactive dashboard with charts and slicers.
- Demonstrate how Microsoft Excel can be used as a business intelligence tool for startups.
